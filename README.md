Proyecto MySQL – Práctica de Base de Datos

Este proyecto consiste en el diseño e implementación de una base de datos relacional utilizando MySQL. Fue desarrollado con el objetivo de afianzar conocimientos en modelado de datos, escritura de consultas SQL y manipulación de información.

La base está estructurada a partir de un Diagrama Entidad-Relación (DER), que define las entidades, atributos y relaciones del sistema.

🎯 Objetivos del proyecto

- Practicar el diseño de bases de datos relacionales
- Desarrollar consultas SQL para distintos escenarios
- Comprender y utilizar relaciones entre tablas (JOINs)

🧱 Estructura del repositorio

El proyecto incluye:

- "DDL-sentencias": creación de tablas, alter table.
- "DML-consultas": select, joins, delete, sum, etc.
- "inserccion-datos": Inserción de datos de prueba generados con IA



✍️ Autor

Proyecto desarrollado como práctica personal para fortalecer habilidades en SQL y bases de datos.
